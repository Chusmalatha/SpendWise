# SpendWise AI — Frontend

This is the React-based frontend for the SpendWise AI audit tool. It is built with **Vite**, **TypeScript**, and **Tailwind CSS**, focusing on a high-fidelity, interactive user experience.

## ✨ Features
- **Multi-Step Audit Form**: A smooth, validation-heavy interface for inputting tool stacks.
- **Glassmorphism Dashboard**: A premium Results Dashboard utilizing Framer Motion for animations.
- **Interactive Data Viz**: Spend breakdowns and savings projections using Recharts.
- **Responsive Layout**: Fully optimized for mobile and desktop viewports.

## 🛠️ Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons (Material Design & Heroicons)
- **Charts**: Recharts
- **Routing**: React Router DOM

## 🚀 Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## 📂 Project Structure
- `/src/components`: Reusable UI components (Navbar, Cards, Modals).
- `/src/pages`: Main application views (Landing, Audit Form, Results).
- `/src/utils`: Helper functions and currency formatters.
- `/src/types`: TypeScript interfaces for the audit data structures.
